Entry 3

Another example which shows a table of items available in the grand exchange. This table shows the current buy/sell price, the ROI%, the buy/sell quantity, among other interesting pieces of information.

The items in the table will be fetched from the OSRS Wiki API 'https://prices.runescape.wiki/api/v1/osrs/' by combining tha mapping query with 'other queries'

Some Notes for implementation (3rd age amulet):

The id-name mapping can be found from here:
https://prices.runescape.wiki/api/v1/osrs/mapping
Then select the item in the list (10344)

Latest high-low prices can be found here:
https://prices.runescape.wiki/api/v1/osrs/latest?id=10344

Daily Trading volume here (updates once per hour):
https://prices.runescape.wiki/api/v1/osrs/volumes
Then select the item in the list (10344)

For timeseries data. Use this:
https://prices.runescape.wiki/api/v1/osrs/timeseries?id=10344×tep=5m
